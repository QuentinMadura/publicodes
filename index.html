Publicodes

Le langage pour les algorithmes d'intérêt public

Découvrir le langage

Publicodes est un langage déclaratif pour modéliser des domaines métiers complexes en les décomposant en règles élémentaires simples.

Il est utilisé par plusieurs administrations pour implémenter des règles de calcul dans des domaines très variés (fiche de paie, empreinte carbone, préavis de retraite).

En plus de calculer des résultats à partir d'une situation saisie, publicodes génère automatiquement une documentation explorable proposant une explication du calcul, qui s'intègre facilement dans une application web. Publicodes est donc particulièrement adapté dans une démarche de transparence des algorithmes publics.

Qui utilise publicodes ?​

  • Mon-entreprise utilise publicodes pour spécifier l'ensemble des calculs relatifs à la législation socio-fiscale en France. Le site permet entre autres de simuler une fiche de paie complète, de calculer les cotisations sociales pour un indépendant ou encore connaître le montant du chômage partiel.
  • Nos Gestes Climat propose un calculateur d'empreinte climat personnel de référence complètement ouvert.
  • Code du Travail Numérique développe un simulateur de préavis de retraite intégrant de nombreuses conventions collectives.
  • Mes Aides Vélos trouve les aides pour acheter un vélo dans votre ville.
  • futur.eco calcule les bilans carbone d'un grand nombre d'activités, plats, transports ou biens.
  • Impact CO2 le site de ressources qui vulgarise et valorise les données environnementales de l'ADEME.
